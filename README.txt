Kabeer's Movies (Version 2.0)
Open-source movie streaming platform built with Next.js, integrating YTX.MX, 
TMDB, and WebTor for seamless movie discovery and streaming. This project aims 
to provide users with a modern, responsive, and feature-rich experience for 
exploring and watching movies, this platform is designed to be scalable, 
user-friendly, and open for community contributions.


Features

- Movie Streaming: Stream movies directly using WebTor and YTX.MX integration.
- Comprehensive Movie Database: Access detailed movie information, including 
ratings, cast, synopsis, and more, powered by TMDB.
- Responsive Design: Built with Next.js for a fast, responsive, and SEO-friendly
user experience.
- Open Source: Contributions are welcome! Help us improve the platform by 
submitting issues, feature requests, or pull requests.

Tech Stack

- Frontend: Next.js (React-based framework)
- Backend: Node.js (API integrations with TMDB, YTX.MX, and WebTor)
- Database: TMDB API for movie data
- Streaming: WebTor and YTX.MX for seamless movie streaming
- Styling: Tailwind CSS for utility-first styling
- State Management: Recoil for efficient state management
- Icons: Heroicons for scalable vector icons
- Deployment: Vercel (or your preferred platform)


Getting Started

To run the project locally, follow these steps:

1. Clone the Repository:
git clone https://github.com/kabeer11000/movies-v2
cd movies-v2

2. Install Dependencies:
npm install

3. Set Up Environment Variables:
Create a .env.local file in the root directory and add your API keys:
TMDB_API_KEY=your_tmdb_api_key

4. Run the Development Server:
npm run dev

5. Open in Browser:
Visit http://localhost:3000 to view the website.
